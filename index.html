Projects

Peeramid | Mar 2023 | (Express, React, HTML5, MongoDB, Javascript and CSS3)
     Live Link
     Peeramid is a fullstack web application where users can rate their day based on Maslow’s hierarchy of needs, and can leave suggestions for other users to improve their score. In this project, I designed and implemented the backend structure for the application using Node.js Express. I also contributed to the construction of middleware to connect the underlying structure to the frontend. The application was completed and deployed within a 4 day time period.

Trailblazer | Feb 2023 | (Rails, React, Redux, HTML5, PostgreSQL, AWS, Javascript and CSS3)
     Live Link
     Trailblazer is a AllTrails clone where users can view trails and parks, leave reviews, and bookmark trails. In this project, I created and deployed a fullstack clone of AllTrails within the time frame of 2 weeks. To supply the application with data, I connected it to an external database and AWS to retrieve images.

Treasure Wars! | Jan 2023 | (Javascript, HTML, CSS)
     Live Link
     Treasure Wars! is a Javascript turn-based strategy browser game in which users attempt to capture each others’ flags. I created the concept and designed the frontend implementation for this game, including game logic, and rendering. The game was deployed within a time frame of 1 week.

Real Time Wind Simulation | Mar 2020 | (C++, OpenGL)
     This is a simulation created for a research paper on aerodynamic force. The simulation demonstrates the change in air density over time utilizing the Navier-Stokes equation. The changes are simulated via color in a jpeg file, with each rgb pixel representing a velocity vector. The equations were implemented using two compute shaders, one for Navier-Stokes and one for diffusion, in OpenGL.

     Formfactor is a semiconductor solutions company located in Livermore, CA. During my internship period there, I was a member of their probe card design department. The tool was successfully deployed at the end of the 3 month internship period.
